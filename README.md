# Набор эффектов CSS

![logo](./images/readme-image.png)

Используемые технологии:
* CSS
* JS
* HTML

Статус: в процессе (видимо, навечно, т.к. всегда есть что добавить)

Задачи:

* создать вариант галереи с приятным эффектом скролла
